- No: 行号
- year: 年份
- month: 月份
- day: 日期
- hour: 小时
- season: 季节
- PM: PM2.5 浓度 (ug/m^3)
- DEWP: 露点 ( 摄氏温度 ) 指在固定气压之下，空气中所含的气态水达到饱和而凝结成液态水所需要降至的温度。
- TEMP: Temperature ( 摄氏温度 )
- HUMI: 湿度 ( % )
- PRES: 气压 ( hPa )c
- cbwd : 组合风向
- Iws : 累计风速 (m/s)
- precipitation: 降水量/时 (mm)
- Iprec : 累计降水量 (mm) mm
